# Survival Arena 传播与商业价值结构

## 一、产品定位

> **一句话：** 局内爽感 + 局外养成 + 社交裂变的 Rogue-like 生存射击游戏

| 维度 | 定位 |
|------|------|
| 核心玩法 | 俯视角生存射击，波次递增，武器自由搭配 |
| 目标用户 | 18-35岁，偏好轻度竞技、收集养成的手游玩家 |
| 对标产品 | 羊了个羊(社交裂变) + Vampire Survivors(局内爽感) + 弓箭传说(养成线) |
| 商业模式 | IAP钻石内购 + 广告变现(激励视频) + 社交裂变获客 |

## 二、传播逻辑（获客裂变）

### 2.1 核心裂变机制：死亡复活

```
玩家阵亡 → 生成专属分享链接(ref=playerId)
         → 分享到微信/QQ
         → 好友点击链接进入游戏
         → 好友获得 +50🪙（新用户奖励）
         → 原玩家获得复活 + 50🪙 + 5💎
         → 好友开始游戏，进入裂变循环
```

**设计要点：**
- 死亡是最强情感触发点 — "就差一点!" → 高分享意愿
- 复活奖励双向互利 — 分享者复活，接收者拿金币
- 首次体验免费（看广告/令牌/好友），降低流失
- 每次分享都是**一次新用户获取**

### 2.2 二级传播机制

| 机制 | 传播节点 | 触发频率 |
|------|---------|---------|
| 💀 死亡复活 | 每局死亡时 | 高频（每局1-3次） |
| 🏆 排行榜段位 | 升段/查看排行时 | 中频（每天1-2次） |
| 🏅 成就解锁 | 达成里程碑时 | 低频但强情感 |
| 🎯 每日挑战 | 达成/超越目标时 | 每天1次 |
| 🔫 武器解锁 | 首次解锁新武器时 | 低频，强炫耀 |

### 2.3 传播漏斗

```
曝光（分享链接）
  → 点击（好友好奇）
    → 进入游戏（无需注册，直接玩）
      → 首次死亡（情感触发）
        → 分享链接（裂变循环）
      → 留存（养成 + 排行榜）
```

**关键指标预估：**

| 指标 | 目标值 | 驱动因素 |
|------|--------|---------|
| 分享率 | 40-60% | 死亡强触发 + 双向奖励 |
| 点击转化 | 15-25% | 好友关系链信任 |
| 新用户留存(D1) | 30-40% | 局内爽感 + 每日奖励 |
| K因子 | 0.3-0.6 | 每个用户带来0.3-0.6个新用户 |

## 三、变现路径（收入模型）

### 3.1 变现架构

```
                    ┌── 激励视频广告（复活/每日奖励翻倍）
用户时间 ──────────┤
                    └── 游戏行为 → 养成进度

                    ┌── ¥6~128 钻石包（核心IAP）
用户付费 ──────────┤
                    └── 💎→🪙 兑换（省时间）
                    └── 💎→⭐ 兑换（局内优势）
```

### 3.2 付费点设计

#### 第一层：便利性付费（💎→🪙）

| 场景 | 痛点 | 付费方案 | 价格感知 |
|------|------|---------|---------|
| 角色升级贵 | 刷金币慢 | 💎→🪙 兑换 | "¥1=200🪙，省2局时间" |
| 差一点升级 | 差几十金币 | 小额兑换5💎=100🪙 | 无痛消费 |

#### 第二层：局内优势（💎→⭐）

| 场景 | 痛点 | 付费方案 | 价格感知 |
|------|------|---------|---------|
| 想用新武器 | 刷星星慢 | 💎→⭐ 开局发放 | "¥2解锁狙击枪" |
| 差一点解锁 | 局内差50⭐ | 小额兑换3💎=50⭐ | 即时满足 |

#### 第三层：身份炫耀

| 场景 | 付费方案 | 价格 |
|------|---------|------|
| 限定皮肤 | 💎购买 | 50-150💎 (¥5-15) |
| 排行榜刷新 | 免费（留存机制） | — |

#### 第四层：广告变现

| 场景 | 广告类型 | 频次限制 | eCPM预估 |
|------|---------|---------|---------|
| 看广告复活 | 激励视频(15s) | 每局1次 | ¥15-30 |
| 每日奖励翻倍 | 激励视频(30s) | 每天1次 | ¥15-30 |
| 死亡后翻倍金币 | 激励视频(15s) | 每局1次 | ¥15-30 |

### 3.3 付费漏斗

```
DAU 1000
  ├── 看广告用户 60% (600人) → 广告收入 ¥9-18/天
  ├── 首充用户 5% (50人) → ¥6×50 = ¥300
  ├── 月卡/重复购买 2% (20人) → ¥30-68/月
  └── 鲸鱼用户 0.5% (5人) → ¥128+/月

预估 ARPU: ¥0.5-1.5/天
预估 ARPPU: ¥15-30/月
```

## 四、留存驱动

### 4.1 短期留存（D1-D7）

| 天数 | 机制 | 效果 |
|------|------|------|
| D1 | 首次死亡 → 免费复活 → 继续玩 | 体验完整循环 |
| D2 | 每日奖励(50🪙) | 登录即得 |
| D3 | 每日奖励 +1💎 | 首次钻石 |
| D5 | 每日奖励 +2💎 | 钻石累计可兑换 |
| D7 | 每日奖励 500🪙+5💎+1🎫 | 周奖励高潮 |

### 4.2 中期留存（D7-D30）

| 机制 | 粘性原理 |
|------|---------|
| 🏆 段位系统 | "差3波升段，再来一局" |
| 🔫 武器解锁 | "攒200⭐买霰弹枪" (2-3局) |
| 🏅 成就收集 | 12个成就，驱动探索 |
| 📜 排行榜 | Top 20 战绩历史，竞争驱动 |
| 🎯 每日挑战 | 随机目标分，达标有奖 |

### 4.3 长期留存（30天+）

| 机制 | 粘性原理 |
|------|---------|
| ⬆️ 属性养成 | 5条升级线，满级需50-80局 |
| 🔫 武器大师 | 4把武器×3条升级线，满级需100+局 |
| 👑 传说段位 | 到达20波 = 终极目标 |
| 🎨 皮肤收集 | 6款皮肤，身份象征 |

## 五、经济循环

```
                 ┌──────────────────────────┐
                 │       用户时间投入         │
                 │  (对局 → ⭐ + 🪙 + 💎)    │
                 └──────────┬───────────────┘
                            │
            ┌───────────────┼───────────────┐
            ▼               ▼               ▼
        ⭐ 星星          🪙 金币          💎 钻石
      (局内消耗)      (全局养成)       (付费/稀有)
            │               │               │
            ▼               ▼               ▼
      武器解锁/升级   角色属性升级     道具/兑换/皮肤
      (跨局保留)     (永久提升)       (便利/炫耀)
            │               │               │
            └───────────────┼───────────────┘
                            │
                            ▼
                  战力提升 → 更高波次
                            │
                            ▼
                  更高排名 → 更强炫耀欲
                            │
               ┌────────────┼────────────┐
               ▼            ▼            ▼
           继续玩        分享炫耀      付费加速
```
