## 1. 核心玩法

### 1.1 基础操作

| 操作 | 按键 | 说明 |
|------|------|------|
| 移动 | WASD | 八方向自由移动 |
| 瞄准 | 鼠标 | 360° 自由瞄准 |
| 射击 | 鼠标左键 | 手动射击 / 自动射击 |
| 闪避 | 空格 | 短距离冲刺，2秒冷却 |
| 暂停 | ESC | 打开暂停菜单 |

### 1.2 游戏循环

```
开始游戏 → 波次递增 → 击杀敌人 → 收集⭐星星
    → 暂停购买武器/道具/升级 → 波次Boss(每5波)
    → 死亡 → 复活/放弃 → 记录战绩 → 养成 → 再来一局
```

### 1.3 敌人系统

| 类型 | 特点 | 首次出现 |
|------|------|---------|
| 小兵 | 基础近战 | 第1波 |
| 跑者 | 高速近战 | 第2波 |
| 坦克 | 高血量 | 第3波 |
| 射手 | 远程攻击 | 第4波 |
| Boss | 每5波，高伤害高血量 | 第5波 |

敌人属性每波递增：HP +8%、速度 +2%、伤害 +5%

## 2. 三种货币经济体系

### 2.1 货币定义

| 货币 | 图标 | 范围 | 获取途径 | 用途 |
|------|------|------|---------|------|
| **星星** | ⭐ | 局内 | 击杀敌人(1⭐, Boss 15⭐)、波次奖励(wave×8)、连杀奖励、钻石兑换 | 购买武器(持久)、武器升级、局内道具 |
| **金币** | 🪙 | 全局 | 每日签到、每日挑战、钻石兑换、好友助力 | 角色属性升级、自动射击、永久道具 |
| **钻石** | 💎 | 全局 | 充值购买、每日奖励(第3/5/7天)、稀有掉落、成就 | 复活令牌、兑换金币/星星、皮肤 |

### 2.2 货币流转

```
        击杀敌人 ──→ ⭐ 星星 ──→ 购买武器(持久)
                         │
        钻石兑换 ─────────┤
                         │
                         └──→ 购买局内道具(本局有效)
                         
        每日签到 ──→ 🪙 金币 ──→ 角色属性升级(永久)
                         │
        钻石兑换 ─────────┤
                         │
                         └──→ 自动射击模块(永久)
                         
        充值 ¥ ──→ 💎 钻石 ──→ 兑换金币/星星
                         │
        成就奖励 ─────────┤
                         │
                         └──→ 复活令牌、皮肤
```

### 2.3 局内外资产统一

- **星星**：局内赚、局内花，但购买的武器**跨局永久保留**
- **金币**：全局持久，所有养成升级永久生效
- **钻石**：全局持久，付费核心货币
- 死亡时显示本局获得星星数，自动记录战绩

## 3. 装备与养成系统

### 3.1 武器系统

| 武器 | 图标 | 解锁费用 | 约合局数 | 特点 |
|------|------|---------|---------|------|
| 手枪 | 🔫 | 免费 | 初始 | 平衡型，1弹丸 |
| 霰弹枪 | 💥 | 200⭐ | 2-3局 | 5弹丸扇形散射 |
| 冲锋枪 | ⚡ | 350⭐ | 3-4局 | 极高射速，弹幕压制 |
| 狙击枪 | 🎯 | 500⭐ | 5-6局 | 3倍伤害，一击制敌 |

- 武器解锁后**永久拥有**，跨局保留
- 可在暂停菜单或主菜单商店中切换装备
- 每局开始时自动装备上次选择的武器

### 3.2 武器升级（每条3路线 × 4武器 = 12条升级线）

| 武器 | 路线1：伤害 | 路线2：射速 | 路线3：特殊 |
|------|-----------|-----------|-----------|
| 手枪 | +8%/级 | +6%/级 | 弹药：+1弹丸/级 |
| 霰弹枪 | +8%/级 | +5%/级 | 精准：-6%散射/级 |
| 冲锋枪 | +7%/级 | +7%/级 | 精度：-8%散布/级 |
| 狙击枪 | +10%/级 | +5%/级 | 穿透：+15%子弹/级 |

- 每条路线最高10级，每级费用 120 + (当前等级-1)×60⭐
- 武器总等级 = 三条路线等级之和÷3（最高Lv.20）
- 升级消耗⭐星星（局内货币）

### 3.3 角色属性升级（消耗🪙金币）

| 属性 | 每级效果 | 基础费用 | 最高等级 | 满级约需 |
|------|---------|---------|---------|---------|
| 最大生命 | +10 HP | 100🪙 | 20 | ~10,500🪙 |
| 移动速度 | +6% | 150🪙 | 15 | ~7,875🪙 |
| 攻击力 | +10% | 150🪙 | 20 | ~15,750🪙 |
| 射速 | +8% | 200🪙 | 15 | ~10,500🪙 |
| 金币加成 | +10% | 100🪙 | 20 | ~10,500🪙 |

- 费用公式：`baseCost × (1 + 等级 × 0.5)`
- 属性永久生效，跨局保留

### 3.4 自动射击模块

- 一次性购买 **400🪙 金币**（永久）
- 效果：武器自动向鼠标方向持续射击
- 跨局保留，无需重复购买

### 3.5 局内道具（消耗⭐星星）

| 道具 | 费用 | 效果 |
|------|------|------|
| 生命恢复 | 30⭐ | 立即恢复30HP |
| 护盾 | 45⭐ | 10秒护盾保护 |
| 速度提升 | 30⭐ | 8秒内移速+50% |
| 双倍伤害 | 60⭐ | 8秒内双倍伤害 |
| 星星磁铁 | 40⭐ | 10秒自动吸取星星 |

## 4. 社交裂变：分享复活

### 4.1 完整流程

```
玩家阵亡 → 死亡界面
  ├── 📺 看广告复活（免费，每局1次，15秒激励视频）
  ├── 🎫 使用复活令牌（每局1次）
  ├── 👥 邀请好友复活（核心裂变机制）
  │     ├── 步骤1：复制链接 / 微信分享 / QQ分享
  │     │     └── 链接包含 ref=playerId 参数
  │     ├── 步骤2：核验分享
  │     │     └── 系统检测好友是否已通过链接进入游戏
  │     └── 核验通过 → 确认复活！
  │           ├── 满血复活
  │           ├── 3秒无敌护盾
  │           ├── +50🪙 金币
  │           └── +5💎 钻石
  ├── 💎 花费5钻石复活
  └── 🏳️ 放弃（记录战绩，返回主菜单）
```

### 4.2 好友侧流程

```
点击分享链接 → 进入游戏（无需注册）
  → 自动检测ref参数 → 获得 +50🪙 金币
  → 开始游戏 → 体验核心玩法
  → 死亡 → 触发分享复活 → 裂变循环
```

### 4.3 核验机制

- 玩家分享后，系统存储 `pendingRevives` 记录
- "核验分享"按钮轮询检查（最多5次，每次间隔800ms）
- 检测到好友点击链接 → 核验通过 → 可复活
- 若未检测到 → 提示返回上一步重新分享

### 5.1 段位系统（5级）

| 段位 | 图标 | 解锁条件 | 颜色 |
|------|------|---------|------|
| 青铜战士 | 🥉 | 默认 | 铜色 |
| 白银骑士 | 🥈 | 到达第5波 | 银色 |
| 黄金勇士 | 🥇 | 到达第10波 | 金色 |
| 钻石英雄 | 💎 | 到达第15波 | 钻石蓝 |
| 传说至尊 | 👑 | 到达第20波 | 红橙 |

- 主菜单显示当前段位 + 进度条
- 死亡界面段位提升提示（"距升段还差X波"）
- 模拟百分位显示（"超越X%的战士"）

### 5.2 成就徽章（12个）

| 类别 | 成就 | 条件 |
|------|------|------|
| 击杀 | 初次狩猎 / 屠戮机器 / 死神降临 | 累计50 / 500 / 2000杀 |
| 波次 | 初露锋芒 / 身经百战 / 无双战将 | 到达5 / 10 / 20波 |
| 得分 | 千分猎手 / 精英杀手 | 单局1000 / 5000分 |
| 连杀 | 连杀高手 / 战场风暴 | 达成10 / 30连杀 |
| 收集 | 武器大师 / 坚持不懈 | 解锁4武器 / 连签7天 |

### 5.3 排行榜界面

- 📊 个人最佳（最高分、最高波次、总击杀、连签天数）
- 📜 战绩记录（Top 20，金银铜标注前三）
- 🏅 成就墙（已解锁亮色，未解锁灰色）
- 🎯 每日挑战（根据水平自动生成分数目标，达成奖励🪙+💎）

### 数据持久化（localStorage）

所有持久数据以 `sa_` 前缀存储在 localStorage：

| Key | 内容 |
|-----|------|
| `sa_coins` | 金币余额 |
| `sa_gems` | 钻石余额 |
| `sa_upgrades` | 角色属性等级 |
| `sa_ownedWeapons` | 武器拥有状态 |
| `sa_weaponUpgrades` | 武器子属性等级 |
| `sa_autoFireOwned` | 自动射击拥有 |
| `sa_equippedWeapon` | 当前装备武器 |
| `sa_highScore` | 最高分 |
| `sa_totalKills` | 累计击杀 |
| `sa_highestWave` | 最高波次 |
| `sa_scoreHistory` | 战绩记录(Top 20) |
| `sa_achievements` | 已解锁成就 |
| `sa_dailyStreak` | 连续签到天数 |
| `sa_dailyTarget` | 每日挑战目标 |
| `sa_pendingStarBonus` | 待发放星星 |
| `sa_reviveTokens` | 复活令牌数 |
| `sa_skins` | 皮肤数据 |

### 浏览器兼容性

- Chrome / Edge / Firefox 最新版
- 需要支持 Canvas、localStorage、requestAnimationFrame
- 移动端支持触屏操作（通过鼠标事件模拟）
