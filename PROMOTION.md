**标题：** 我写了一个网页射击生存游戏，纯前端+Supabase后端，来试试你能撑到第几波？

**正文：**

周末手搓了一个网页射击小游戏——生存竞技场（Survival Arena）。

玩法：WASD移动，鼠标瞄准射击，撑过一波又一波敌人，每5波出一个BOSS。

技术栈：
- 纯原生 JS + Canvas，没有用任何游戏引擎
- 零依赖，所有代码手写（连 Supabase 都是用 REST API 直连，没装 SDK）
- 好友复活系统（基于 Supabase 实时验证，支持跨设备）
- 武器升级、皮肤、每日签到、成就系统
- 完整的数据分析埋点，dashboard 可以看所有玩家数据

链接：https://aaa-geeker.github.io/Survival-Arena---/

GitHub 开源：https://github.com/AAA-Geeker/Survival-Arena---
